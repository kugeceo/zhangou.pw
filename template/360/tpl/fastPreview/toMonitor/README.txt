Axiplayer

Version: 3.0
January 27, 2020
Created by Axigear (www.axigear.ru)

Axiplayer is HTML web app, contains a 360-degree product photo.
This app is used to integrate 360-degree photos on the site by using <iframe> html tag.

Axiplayer - созданный на языке HTML проигрыватель фотографий в формате 360 градусов.
Приложение используется для интеграции таких фотографий на сайты используя тег <iframe>.
